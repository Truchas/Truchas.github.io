Patch Algorithms¶

genre includes three patching algorithms.

  • Vertex Anchor Cluster Paving (PAVE)
  • Vertex Anchor Cluster (VAC)
  • Variational Shape Approximation (VSA)

Warning

The VAC and VSA algorithms are experimental and not recommended for regular users. For best results, use the PAVE algorithm.

Result of running PAVE (left), VAC (center),and VSA (right) on the ‘basic hemi’ enclosure.

PATCHES Namelist¶

The PATCHES namelist defines the parameters used by the patching algorithms. The namelist supports many parameters, but not all parameters are used by all algorithms. Parameters only used by a particular algorithm are prefixed with the algorithm’s name.

&PATCHES
  patch_algorithm = 'PAVE'
  verbosity_level = 3
  max_angle = 30.0
  pave_split_patch_size = 4
/

Example PATCHES namelist

The PATCHES namelist parameters include:

  1. PATCH_ALGORITHM: Patch algorithm to execute.

  2. VERBOSITY_LEVEL: Verbosity level of the patch algorithm.

  3. MAX_ANGLE: Maximum angle between adjacent faces

  4. PAVE_MERGE_LEVEL: Controls the aggressiveness of patch merging

  5. PAVE_SPLIT_PATCH_SIZE: Split patches with up to this number of faces

  6. PAVE_RANDOM_SEED: Sets random number generator seed.

  7. VAC_MERGE_LEVEL: Controls the aggressiveness of patch merging

  8. VAC_SPLIT_PATCH_SIZE: Split patches with up to this number of faces

  9. VSA_MAX_ITER: Maximum iterations for the VSA algorithm

  10. VSA_MIN_DELTA: Minimum change in patch proxies for the VSA algorithm

  11. VSA_AVG_FACES_PER_PATCH: Average faces per patch for the VSA algorithm

  12. VSA_RANDOM_SEED: Sets random number generator seed.

See also

For detailed information on these parameters, refer to the PATCES namelist documentation.
