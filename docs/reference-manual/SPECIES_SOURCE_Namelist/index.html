29. SPECIES_SOURCE NamelistÔÉÅ

The SPECIES_SOURCE namelist is used to define external volumetric sources for the species advection-diffusion model.

Namelist Usage

Required/Optional:

Optional

Single/Multiple Instances:

Multiple

29.1. Namelist VariablesÔÉÅ

  • comp_id

  • cell_set_ids

  • source

  • source_func

comp_idÔÉÅ

The index of the species component to which this source applies.

Type:

integer

Default:

1

Note:

The number of species components is defined by the PHYSICS namelist variable number_of_species.

cell_set_idsÔÉÅ

A list of cell set IDs that define the subdomain where the source is applied.

Type:

a list of up to 32 integers

Default:

none

Valid Values:

any valid mesh cell set ID

Note:

Different instances of this namelist for a given component must apply to disjoint subdomains; overlapping of source functions is not allowed.

ExodusII mesh element blocks are interpreted by Truchas as cell sets having the same IDs.

sourceÔÉÅ

The constant value of the source. To specify a function use source_func instead.

Type:

real

Default:

none

source_funcÔÉÅ

The name of a FUNCTION namelist that defines the source function. The function is expected to be a function of \((t,x,y,z)\).

Type:

string

Default:

none
