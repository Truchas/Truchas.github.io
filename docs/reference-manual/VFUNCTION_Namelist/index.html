32. VFUNCTION Namelist¶

32.1. Overview¶

Similar to the FUNCTION namelist, the VFUNCTION namelist is used to define a vector-valued function that can be used in some situations where vector-valued data is needed, such as the specification of the boundary velocity in a flow boundary condition.

These are general vector-valued functions of one or more variables, \(y = (y_1,...,y_m) = f(x_1,...,x_n)\). The dimension of the value, the number of variables, and the unknowns they represent (i.e., time, position, temperature, etc.) all depend on the context in which the function is used, and that is described in the documentation of those namelists where these functions can be used. Currently only a single function type can be defined:

Tabular Function. This is a continuous, single-variable function \(y=f(s)\) linearly interpolated from asequence of data points \((s_i,y_i), i= 1,...,p\), with \(s_i \lt s_i+1\). The variable \(x_d\) that is identified with \(s\) is specified by tabular_dim.

32.2. VFUNCTION Namelist Features¶

Required/Optional : Optional
Single/Multiple Instances: Multiple

32.3. Components¶

  • Name

  • Type

  • Tabular_Data

  • Tabular_Dim

32.3.1. Name¶

Description : A unique name by which this vector function can be referenced by other namelists.
Type : A case-sensitive string of up to 31 characters.
Default : None

32.3.2. Type¶

Description : The type of function defined by the namelist.
Type : case-sensitive string
Default : None
Valid values: tabular for a tabular function

32.3.3. Tabular_Data¶

Description : The table of values \((s_i,y_i)\) defining a tabular function \(y=f(s)\). Use Tabular_Dim to set the variable identified with \(s\).
Type : real array
Default : None
Valid values: This is a \((m+ 1) × p\) array that is most easily specified in the following manner
\[ \begin{align}\begin{aligned}Tabular\_Data(:,1) =s_1, y_{11}, ..., y_{m1}\\Tabular\_Data(:,2) =s_2, y_{12}, ..., y_{m2}\\ .\\ .\\ .\\Tabular\_Data(:,p) =s_p, y_{1p}, ..., y_{mp}\end{aligned}\end{align} \]

32.3.4. Tabular_Dim¶

Description : The dimension in the \(m\)-vector of independent variables that serves as the independent variable for the single-variable tabular function.
Type : integer
Default : 1
